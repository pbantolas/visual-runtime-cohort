# MoltenVK is a macOS smoke-test path

The Vulkan backend is being introduced to support an eventual Linux harness, while MoltenVK on macOS is only an instructor-side smoke-test path for early validation. The macOS harness may be used to prove the Vulkan backend can render, but the project should not treat Vulkan-on-macOS as a required course path or a hard dependency for normal macOS users.
